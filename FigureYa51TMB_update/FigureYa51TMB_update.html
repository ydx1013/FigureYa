Use the expression of a particular gene to group, draw a TMB box plot with scatter points for each group, and calculate the p value.

Application scenario

TMB

TMB

TMB

With the help of mutation data TMB, link your genes to immunotherapy prognosis.

You can group based on high or low expression of a particular gene (or group based on other information) and compare the TMB values of different groups.

In the next step, it is also possible to do survival analysis by grouping with high or low TMB values.

ggplot2ggpubrp value

Use ggplot2 to draw the graph, use ggpubr to calculate the p value.

mycol <- c("darkgreen", "darkorchid3", "orange") # consistent with the number of groups
p <- ggplot(TMB_clinical_mRNA, aes(x = Expr_level, y = TMB, color = Expr_level)) +
  geom_boxplot(outlier.color = NA) + # hide the abnormal points on the box plot

  scale_color_manual(values = mycol) + # custom color scheme
  
  stat_compare_means(#paired = T, #whether you want a paired test
                     #wilcox.testt.test comparison between two groups using wilcox.test or t.test
                     method = "wilcox.test", 
                     
                     #kruskal.testanova multiple group comparisons using kruskal.test or anova
                     #method = "kruskal.test",
                     label.y = max(TMB_clinical_mRNA$TMB)*1.1) + #label the position of label
  
  theme_bw() + # remove background color
  theme(panel.grid =element_blank()) + # remove gridlines
  theme(panel.border = element_blank()) + # remove outer border
  theme(axis.line = element_line(colour = "black")) + # display line along coordinate axis
  xlab("Discovery Set") + 
  #ylim(0,5) + #Y set the Y-axis range
  guides(color=FALSE) # legend is not displayed
